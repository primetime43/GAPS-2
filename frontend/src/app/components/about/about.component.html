GAPS 2 is a rewrite of the original GAPS project, now written in Python instead of Java. GAPS finds movies you're missing in your media server (Plex, Jellyfin, or Emby). It's a great way to find additional movies that you might be interested in based on collections from your library.

The GAPS 2 project aims to bring the same functionality with the simplicity and versatility of Python.

Gaps searches through your media server library. It then queries for known movies in the same collection using The MovieDB, which you will need to get. Don't worry, we'll walk you through those steps. Anyway, if those movies don't exist in your library, Gaps will recommend getting those movies, legally of course.

An example would be having a copy of 'Alien (1979)' and Gaps recommending 'Aliens (1986)' and 'Alien 3 (1992)' to be added to your collection.
